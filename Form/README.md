# Sports Carnival Registration

A modern, responsive web application for registering participants in a sports carnival event. Features a dynamic image slider, an engaging introduction section, and a user-friendly registration form with client-side validation.

## Features

- **Responsive Design**: Adapts seamlessly to mobile, tablet, and desktop devices.
- **Image Slider**: Showcases event highlights with smooth transitions and navigation dots.
- **Interactive Form**: Collects participant details with input validation, including student ID, password, academic program, and more.
- **Activity Highlights**: Displays event activities with icons and hover effects.
- **Accessible UI**: Clean, intuitive interface with clear labels and hints.
- **Stylish Animations**: Includes hover effects, transitions, and a gradient-themed design.

## Technologies Used

- **HTML5**: Semantic structure for the webpage.
- **CSS3**: Custom styles with Flexbox, gradients, and responsive media queries.
- **Font Awesome**: Icons for activities and form elements.
- **No JavaScript**: Pure HTML/CSS solution for simplicity and performance.
