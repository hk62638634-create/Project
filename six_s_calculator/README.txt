# 6S Audit Calculator

## Overview
Developed in 2025, this tool automates the analysis of 6S audit data (Sort, Set in Order, Shine, Safety, Standardize, Sustain), processing over 500,000 records from Excel and generating professional Word reports. It was recognized for its innovative approach in 2025.

## Features
- Loads audit data from Excel files, supporting multiple locations.
- Calculates scores for 6S categories (e.g., Sort, Shine) with detailed breakdowns.
- Provides a user-friendly Tkinter GUI for non-programmers.
- Exports formatted Word reports with summary tables and comments.

## Tech Stack
- Python
- Pandas
- Tkinter
- python-docx
- openpyxl

## Files
- [six_s_calculator.py](six_s_calculator.py): Main script implementing all features.
- [requirements.txt](requirements.txt): List of Python dependencies.

## Installation and Usage
1. Install Python 3.8+.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the script: `python six_s_calculator.py`.
4. Select an Excel audit file and click "Calculate 6S Scores" or "Export Word Report".
